# Атомарное чтение из n ячеек

По аналогии с Read + Write, рассмотрим ситуацию, когда при выходе из бивалентной конфигурации поток 1 (не умаляя общности) читает n ячеек. Запустим из этого состояния 2 поток, его поведение будет такимже, как при выходе из бивалентного состояния, так как конфигурации не отличаются. Отсутствие различия приводит к противоречию с существование бивалентного состояния. Получается, задача свелась к Read + Write, а их $`ConsensusNumber=1`$.
